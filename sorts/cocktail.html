Summary

Cocktail Sort is also known as bidirectional Bubble Sort, Cocktail Shaker Sort, Shaker Sort, Ripple Sort, Shuffle Sort, or Shuttle Sort. This sort is named mostly for the way that the data moves. Data is shuttled to either the top or the bottom as the sort works it's way up or down the values. This sort fixes the problem of turtles in Bubble Sort.

Optimization

As stated above Cocktail Sort is fundamentally an optimization of Bubble Sort. So the easiest way to understand this optimization is to compare the two.

Cocktail without a short circuit takes the same amount of time as a Bubble Sort without a short circuit that skips sorted elements. However, in the case that the array has turtles the Cocktail Sort with a short circuit will be much faster than a Bubble Sort with a short circuit. This is because cocktail can move values down at the same speed it can move values to the top. This allows it to handle more kinds of value sets than Bubble Sort. For example, if only the first and last element of an array have been switched Cocktail can move the last element to the end and the first element to the beginning in one set of moving up and down the array.
