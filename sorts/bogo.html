Summary

Bogo Sort is a highly ineffective sort. It's mostly a joke sort. The idea is that you take an array and shuffle then check if it's in order. If it is then you're done, if not shuffle it again. This is one of many sorts that have been created to be purposefully ineffecient while still being correct.

Versions

There are a lot of "versions" of Bogo Sort. Wikipedia lists Permutation Sort, Stupid Sort, Slowsort, Shotgun Sort or Monkey Sort as other names for Bogo Sort. It also lists Gorosort, Bogobogosort, Bozosort, and Worstsort as related algorithms.

Bozosort works like bogo sort, but it only swaps two random elements at time. This seems like it might work better as it won't switch the array from almost sorted to completely unsorted in one move. However, as it still is choosing to swap elements randomly it could still go on forever.

Stupid Sort was the original name of Gnome Sort. However, it has also become an alternative name for Bogo Sort.

Slowsort is in fact another sorting algorithm. So while Bogo is sometimes called that it is in fact a misnomer.
