Summary

Cocktail Sort is also known as bidirectional Bubble Sort, Cocktail Shaker Sort, Shaker Sort, Ripple Sort, Shuffle Sort, or Shuttle Sort. This sort is named mostly for the way that the data moves. Data is shuttled to either the top or the bottom as the sort works it's way up or down the values. This sort fixes the problem of turtles in Bubble Sort.

Optimization

As stated above Cocktail Sort is fundamentally an optimization of Bubble Sort. So the easiest way to understand this optimization is to compare the two.

Cocktail without a short circuit takes the same amount of time as a Bubble Sort without a short circuit that skips sorted elements. However, in the case that the array has turtles the Cocktail Sort with a short circuit will be much faster than a Bubble Sort with a short circuit. This is because cocktail can move values down at the same speed it can move values to the top. This allows it to handle more kinds of value sets than Bubble Sort. For example, if only the first and last element of an array have been switched Cocktail can move the last element to the end and the first element to the beginning in one set of moving up and down the array. While Bubble Sort will only be able to move the turtle one index each round.

Comparing Efficency Among Orderings

Cocktail like Bubble is very quick on an ordered array. It is also quick on arrays where only a couple elements are out of place. However, unlike bubble we see this efficency in both directions. This allows it to be slightly more efficent on a random list. However, a reversed list will take the max time possible every time. This is because every element has to be moved into position.
